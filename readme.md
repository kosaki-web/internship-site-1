# 伝統工芸品セレクトショップ「F.K. CODING & DESIGN」LP制作（モダンバージョン）

実在する店舗の移転およびサイトリニューアルに向け、AI技術を実務プロセスに組み込んで制作したトップページのコンセプトモデルです。
「奈良の伝統を、今の暮らしに。」をテーマに、伝統工芸品が持つ本質的な美しさと品格を引き立てる、落ち着きのあるデザインを目指しました。

クライアントの『振り切ったポップさも見たい』という要望に応えたのが【ポップ案】で、ブランドの伝統を堅実に守る方向性を追求したのが【本案】です。

## 👤 ターゲット層

- **メイン層**: 本物志向の奈良伝統工芸品を求めている観光客。

## 🎨 コンセプト

- **「伝統に寄り添う、シンプルさと静謐さ」**
- 奈良の伝統工芸品（赤膚焼、一刀彫、仏像など）の各セクションが最も際立つよう、装飾を最小限に抑えたミニマルなレイアウトを採用しました。
- 企業実習の担当者から提示されたプロンプト案をベースに、さらにプロンプトを加えて細かく調整。「伝統工芸店としての信頼感」を重視したトーン＆マナーに仕上げています。

## ✨ 実装・こだわったポイント

### 1. 職人魂と誠実さを伝えるビジュアル生成

- **Hero画像の再定義**: 奈良の伝統工芸品と、それらを真剣に取り扱う店舗の姿勢が少しでも伝わるよう、生成AI（Gemini）へのプロンプトを厳選。インパクトがありつつも、店舗の「格」を損なわない象徴的なビジュアルを構築しました。
- **商品画像のトーン＆マナー管理**: AIで生成した写実的な商品画像（赤膚焼、奈良漆器など）がバラバラの雰囲気にならないよう、Photoshopを用いて色調や明暗を一つひとつ調整。サイト全体に一貫した「静かな空気感」を持たせました。

### 2. 伝統美を引き立てるタイポグラフィと設計

- **パフォーマンス最適化**: 次世代画像形式のWebPを採用し、`picture`タグによるフォールバック設定や`loading="lazy"`を実装。リッチな画像を多用しつつ、観光客のモバイル閲覧でもストレスのない表示速度を意識しました。

### 3. 実務を想定したコンポーネント実装

- **レスポンシブ対応**: スマートフォンでの閲覧時も品格を損なわないよう、JavaScript（Vanilla JS）による独自の「menu-trigger（ハンバーガーメニュー）」を実装。
- **ナビゲーションの利便性**: ページ上部へスムーズに戻る「js-pagetop」ボタンを設置し、商品情報の比較検討がしやすい回遊性を確保しました。

## 🛠 使用ツール・技術

- **言語**: HTML5, CSS3, JavaScript (Vanilla JS)
- **デザイン・編集**: Photoshop (AI生成画像のレタッチ・一括補正)
- **AI活用**: Gemini (旧サイト要約、セクション構成案、デザイン・画像生成プロンプトの調整)
- **エディタ**: VS Code
- **外部リソース**: Google Fonts

## 📝 制作の背景と課題解決

職業訓練校の企業実習にて制作。店舗移転に伴う「新サイト制作の参考用プロトタイプ」というミッションに対し、AIをフル活用して短期間で高精度なビジュアル提案を行いました。
「指示された項目を網羅する」という要件を満たした上で、AI生成物に人間の手で「一貫性」と「店舗の想い」を吹き込むプロセスを重視しました。
